# ElixNode — Industrial Digital Twin Safety Monitor

## Quick Start

### 1. Backend (server/)

```powershell
cd server
npm install
node index.js
```

### 2. Frontend (client/)

```powershell
cd client
npm install
npm run dev
```

## Tech Stack

- Frontend: React + Vite + Recharts + Zustand
- Backend: Node.js + Express + WebSocket
- Database: PostgreSQL (Aiven Cloud)
- AI: Google Gemini 1.5 Flash
- Alerts: EmailJS
- Data: HAI SCADA Dataset
